# Vision Assistant

Route: `/vision`

Vision Assistant is a browser-based OCR tool that extracts searchable text from uploaded images.

![Vision Assistant](https://github.com/scidsg/hushline-screenshots/blob/main/releases/latest/admin/auth-admin-tools-vision-assistant-desktop-light-fold.png?raw=true)

## How It Works

- Upload one or more images
- Hush Line runs OCR in the browser
- Extracted text is displayed for review and copy/paste

## Intended Use

This is useful when disclosures include photos of screens, documents, or messages and investigators need searchable text.

## Privacy

- OCR processing runs client-side in the browser.
- No external API calls are required for extraction in this workflow.
