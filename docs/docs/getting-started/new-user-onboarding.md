# New User Onboarding

New user onboarding is a guided, four-step process that will result in a fully functional account that's ready to receive secure, anonymous tips.

## Step 1. Add Information About Yourself

Add a clear bio, extra links, or your Signal username to help your community.

![Settings](https://github.com/scidsg/hushline-screenshots/blob/main/releases/latest/newman/auth-newman-onboarding-profile-desktop-light-fold.png?raw=true)

## Step 2. Enable End-to-End Encryption

Add an encryption key so you can receive messages securely:

![Encryption settings wireframe](https://github.com/scidsg/hushline-screenshots/blob/main/releases/latest/newman/auth-newman-onboarding-encryption-desktop-light-fold.png?raw=true)

## Step 3. Enable Message Forwarding

Forward messages so you don’t have to log back in. If you imported a key from Proton in the previous step, you should use the same email address here.

![Email](https://github.com/scidsg/hushline-screenshots/blob/main/releases/latest/newman/auth-newman-onboarding-notifications-desktop-light-fold.png?raw=true)

## Step 4. Add Yourself to the User Directory

Opt-in to the public user directory so whistleblowers can find your account.

![Directory](https://github.com/scidsg/hushline-screenshots/blob/main/releases/latest/newman/auth-newman-onboarding-directory-desktop-light-fold.png?raw=true)

## Skipping Onboarding

If you skip onboarding, you can easily access it again using the **Account Setup** link in the header.

![Empty Inbox](https://github.com/scidsg/hushline-screenshots/blob/main/releases/latest/newman/auth-newman-inbox-desktop-light-fold.png?raw=true)
